🔐 Advanced Password Generator
A Python application that generates strong and secure passwords based on user preferences.

✨ Features
Choose password length
Include uppercase letters
Include lowercase letters
Include numbers
Include special symbols
Ensures a strong random password
Generate multiple passwords in one session
Input validation for user-friendly experience
🛠️ Technologies Used
Python 3
random module
string module
🚀 How to Run
Clone this repository:

Open the project folder.

Run:
python password_generator.py
